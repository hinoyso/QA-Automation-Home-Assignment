# QA Automation Home Assignment

This project contains the solution for the QA Automation home assignment.
It includes:

- A full end-to-end automation flow using Playwright (JavaScript)
- A clear Page Object Model (POM) structure
- A detailed Test Standard (manual test case)

# How to Install & Run 
1. Install dependecies: npm install
2. Run all tests: npx playwright test
3. Run in debug mode: npx playwright test --debug

# Test Case 
The full Test Standard document is included in the project:

QA_Test_Standard.pdf

# Assignment Requirements Coverage
This project implements all the required steps:

- New user registration
- Redirect and header validation
- Navigating to Digital Downloads
- Choosing a random product
- Adding it to the cart
- Verifying the product name matches in the cart
